Данная таска основана на коде предыдущей таски
Задача:
 1. Внедрить Vuex вместо provide/inject и props/emit 
 2. Из компонентов все получаем с помощью getters и изменяем с помощью actions (которые в конце вызывают mutations).
 3. Добавить возможность изменения созданного контакта.
 4. Удаление контакто должно происходить асинхронно (то есть с задежкой например 3 секунды), нажимете на кнопку через три секунды контакт удаляется.
 5. Выделить отдельный модуль в store для контактов и использовать в нем namespace.
 6. Использовать хелперы от vuex (mapGetters, mapActions ... )
